# python-game-gif

## Usage
### Running App

`$ python3 app.py [--port 8080] [--bind 127.0.0.1] [--css default.css]`

### Creating Request
`http://[url]:[port]/[gameid].gif`

Example
`http://127.0.0.1:8080/nyMfOUDN.gif`

## Installation
`pip3 install -r requirements.txt`
